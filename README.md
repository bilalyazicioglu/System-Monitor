# System Monitor

Terminal tabanlı sistem izleme uygulaması.

## Teknolojiler

- [Bubble Tea](https://github.com/charmbracelet/bubbletea) - Terminal UI framework
- [gopsutil](https://github.com/shirou/gopsutil) - Sistem bilgileri

## Kurulum

```bash
go mod tidy
go run .
```

## Kullanım

- `q` veya `Ctrl+C` ile çıkış yapın
